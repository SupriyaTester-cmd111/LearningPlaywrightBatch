# Learning Playwright Batch

This project is designed to help you learn and practice using Playwright for browser automation and testing.

## Project Structure

```
LearningPlaywrightBatch
├── tests
│   └── 01_Basics.js
├── .gitignore
├── package.json
├── playwright.config.js
└── README.md
```

## Files Overview

- **tests/01_Basics.js**: Contains a simple JavaScript code that logs "Hello Sharvil" to the console.
- **.gitignore**: Specifies files and directories that should be ignored by Git, such as `node_modules`, logs, and environment files.
- **package.json**: Configuration file for npm, listing dependencies, scripts, and metadata for the project.
- **playwright.config.js**: Contains configuration settings for Playwright, including test directory, timeout settings, and browser configurations.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/SupriyaTester-cmd/LearningPlaywrightBatch.git
   ```

2. Navigate to the project directory:
   ```
   cd LearningPlaywrightBatch
   ```

3. Install dependencies:
   ```
   npm install
   ```

## Usage

To run the tests, use the following command:
```
npx playwright test
```
